# Balatro Game Effects Documentation

Complete documentation of all game effects extracted from Balatro source code.

This repository contains comprehensive documentation of:
- **Enhancements** - Card enhancements (Bonus, Mult, Wild, Glass, Steel, Stone, Gold, Lucky)
- **Editions** - Card editions (Base, Foil, Holographic, Polychrome, Negative)
- **Vouchers** - Permanent shop upgrades (32 total, base and upgraded)
- **Consumables** - Tarot cards, Planet cards, and Spectral cards
- **Seals** - Card seals (Gold, Red, Blue, Purple)
- **Tags** - One-time effects (24 types)
- **Blinds & Bosses** - All enemy types and their debuffs (30 total)
- **Challenges** - Special game modes (20 challenges)
